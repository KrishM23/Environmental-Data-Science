Build something real, start to finish

Each project uses a real dataset from our library. Beginner projects include runnable notebooks; video walkthroughs are on the way. No paid software required.

Colab needs a public GitHub repo or a deployed site URL in assets/site-config.js. If Colab opens blank, check repo visibility under GitHub Settings.
