# Terrain — Environmental Data Science at UCLA

Terrain is a student-built entry point for environmental data science at UCLA—from students, for students. It exists because getting started meant hunting across `.gov` portals, faculty pages, and random GitHub repos with no clear path from “curious” to “I built something.”

Everything is free and runs on free tools. No paid software, no powerful laptop required, and beginner projects assume no prior coding experience.

## What's on the site

- **Datasets** — 11 curated environmental datasets (CalEnviroScreen, EPA AirNow, NOAA, NASA Earthdata, ICESat-2, Climate TRACE, USGS, UCLA emissions, and more), each with a plain-English explainer.
- **Starter projects** — 10 guided projects in R or Python, from beginner to advanced. Each uses live or public data and ends with something real: a map, chart, or model. Notebooks are runnable in Colab or RStudio.
- **Faculty** — UCLA researchers across departments who work in environmental data science.
- **Programs** — Campus pathways (IoES, fellowships, capstones, and more).
- **Carbon tracker** — Live chart of UCLA’s verified emissions vs. UC’s 90%-by-2045 commitment.

## Who it's for

Students who want to do real environmental work but don't know where to start—especially first-gen, low-income, and underrepresented students without an existing network into research.

## Built with

Plain HTML, CSS, and JavaScript. Chart.js for the tracker. Static site—fast, free to host, easy to contribute to.

## Status

Live and usable: explainers, datasets, project library, faculty directory, and notebooks. **Mentor walkthrough videos** are the main piece still in progress.
